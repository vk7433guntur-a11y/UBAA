# 更新日志

UBAA 的版本号由 `gradle.properties` 管理，发布资产通过 GitHub Release 工作流生成。客户端版本检查接口为 `GET /api/v1/app/version`。

所有版本的完整发布说明和下载资产请参见 [GitHub Releases](https://github.com/BUAASubnet/UBAA/releases)。
